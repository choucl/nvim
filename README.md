# Neovim Configuration of Yuyu's

## Dependencies

* Nodejs
* npm
* pynvim
* ranger

## Command To Execute After Clone

* To install plugins
    ```
    :PlugInstall
    ```
* To install coc extensions

    ```
    :CocInstall coc-json coc-python coc-vimlsp coc-clangd
    ```
* To install clangd

    ```
    :CocCommand clangd.install
    ```
